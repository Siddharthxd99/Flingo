## Overview

Flingo is a comprehensive developer tool designed to enhance Discord bot development with seamless integrations, dynamic interactions, and robust moderation features. It provides a modular architecture that simplifies content delivery, security management, and user engagement within your applications.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python
- **Package Manager:** Conda

### Installation

Build Flingo from the source and install dependencies:

1. **Clone the repository:**

```bash
git clone https://github.com/Siddharthxd99/flingo
```

2. **Navigate to the project directory:**

```bash
cd flingo
```

3. **Install the dependencies:**

```bash
pip install -r r.txt
```

### Usage

Run the project with:

Using **python:**

```bash
python -m venv .venv
.venv\Scripts\Activate.ps1
```
Run:
```bash
python main.py
```
